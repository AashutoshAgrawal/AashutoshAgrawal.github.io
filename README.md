## ğŸš€ Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/AashutoshAgrawal/AashutoshAgrawal.github.io.git

# Navigate to project directory
cd AashutoshAgrawal.github.io

# Install dependencies
npm install
```

### Development

```bash
# Start development server (runs on http://localhost:8080)
npm start

# Or use the dev command
npm run dev
```

The site will automatically reload when you make changes.

### Building for Production

```bash
# Build the project
npm run build

# Preview production build
npm run preview
```

The optimized build will be in the `dist/` folder, ready for deployment.

### Testing

```bash
# Run tests
npm test
```

## ğŸ¨ Customization

### Updating Personal Information

Edit `src/constants/index.ts` to update:
- Personal info (name, title, bio, contact)
- Social media links
- Projects and experience
- Skills and testimonials

### Theme Configuration

Dark theme is set as default. Modify `src/contexts/ThemeContext.tsx` to change theme behavior.

### Email Configuration

Set up EmailJS by following the guide in `EMAILJS_SETUP_GUIDE.md`.

### Analytics

Configure Google Analytics by following `ANALYTICS_SETUP.md`.
